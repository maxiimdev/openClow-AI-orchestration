# AGENTS

## Роли v1 (минимум)

### 1) Orchestrator (OpenClaw)
**Ответственность:**
- intake задачи
- уточнение требований
- планирование и маршрутизация
- контроль статуса пайплайна

**Не делает:**
- не пушит напрямую в `main`
- не раздувает процесс лишними промежуточными агентами

### 2) Claude Implementer
**Ответственность:**
- реализация изменений в коде
- соблюдение проектных правил
- атомарные коммиты

### 3) Claude Reviewer
**Ответственность:**
- критическая проверка кода (качество/риски)
- поиск дефектов и уязвимостей
- конкретные замечания и фиксы

### 4) Claude Test Engineer
**Ответственность:**
- тестовый прогон/чеклист
- фиксация непройденных кейсов

## Замечание
На старте эти роли могут быть разными режимами одного Claude Code через bridge, а не отдельными сложными агентами.
