# AI Orchestration Docs (v1)

Эта папка — рабочая документация по системе оркестрации разработки.

## Состав
- `SYSTEM_OVERVIEW.md` — цель, архитектура и границы системы
- `FLOW.md` — статусная модель и жизненный цикл задачи
- `AGENTS.md` — роли, ответственность и правила масштабирования
- `EXPERT_PACKS.md` — модель экспертности и структура пакетов ролей
- `TOKEN_POLICY.md` — принципы экономии токенов и операционные лимиты
- `DECISIONS_LOG.md` — журнал принятых решений и причин

## Как пользоваться
1. Начинать с `SYSTEM_OVERVIEW.md`.
2. Для операционной работы держать открытым `FLOW.md`.
3. При изменениях процесса сначала обновлять docs, потом реализацию.
4. Любое значимое решение фиксировать в `DECISIONS_LOG.md`.

## Принцип
Документация должна быть:
- прикладной;
- актуальной;
- проверяемой на реальных задачах.

Если документ перестал помогать в работе — его нужно упростить или переписать.
