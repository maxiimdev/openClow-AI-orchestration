# HEARTBEAT.md

## Daily continuity tasks (UTC)

1. If date changed since last note, create/update `memory/YYYY-MM-DD.md` with:
   - key decisions
   - active tasks/branches
   - risks/blockers
   - next step for tomorrow

2. Update persistent project memory when major changes happen:
   - `memory/project-congress2026.md`

3. If no meaningful changes since last heartbeat, do nothing.

4. In quiet hours (23:00–08:00 UTC), only write memory if there was major work.
